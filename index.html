Main Features

Built on GCP

The toolset leverages the Google Cloud Platform including one of the most powerful data warehouse platforms in the world, BigQuery.

Javascript

The code-base is completely Javascript, the world's most popular and most contributed to open source code language.

Highly Extensible

The toolset is built for integrating all types of data sources and targets. New sources and targets can be integrated without downtime.

Serverless

All processing is done using cloud-based managed services. No servers, no virtual machines, no problems.

Scales Automatically

From a handful to tens of thousands of requests, the system automatically scales to meet demand without user intervention.

Open Source

The code is open source, MIT-licensed, and can be used be anybody, anywhere.

Why we need a new ETL toolset.

As companies are pressured by industry experts to adopt data-first strategies such as AI and machine learning they face the largest struggle: centralizing and maintaining structured data. Most commercial BI and data processing tools overpromise and can cost millions in licensing and implementation with lack-luster results. MicroETL leverages the language with that largest community of developers and provides convenience packages to reduce development times. Open source ensures access to the most cutting-edge software is available to organizations of any size.

Components

API

Highly-available, authenticated endpoints for incoming data.

Data Lake

Raw data storage, always available, and automatic cold-storage partitioning.

Transformers

Cloud Functions that manipulate raw data with a high-degree of control.

Fetchers

Scheduled or triggered functions that retrieve data from external systems.

Pushers

Cloud Functions that propagate data across external systems.

Data Warehouse

A dataset that provides a single source of truth across the organization.
